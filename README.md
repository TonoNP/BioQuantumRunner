# BioQuantum Runner

Ecosistema personal para análisis de entrenamiento (Polar/Garmin), rendimiento y modelos IA, con una capa de ideas inspiradas en información cuántica.

## Estructura
- data/       Datos crudos y procesados
- notebooks/  Análisis en Jupyter
- src/        Código Python (módulos)
- reports/    Salidas (PDF/figuras/resúmenes)

## Entorno
Este proyecto usa el entorno conda: `bioquantum` (Python 3.11).
